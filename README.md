# Student Mark Analysis using Python

## Overview
This project performs basic analysis of student marks using Python, Pandas, and Matplotlib.  
It is designed to run easily in Google Colab or Jupyter Notebook.

## Files Included
- `student_marks.csv` → Dataset containing student marks
- `colab_student_mark_analysis.ipynb` → Google Colab notebook for analysis
- `README.md` → Project documentation

## Features
- Load and analyze student mark data
- Calculate:
  - Total marks
  - Average marks
- Identify top-performing student
- Visualize subject-wise average marks using a bar chart

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset Details
The dataset contains:
- Student ID
- Student Name
- Maths Marks
- Science Marks
- English Marks

## How to Run in Google Colab

### Step 1: Open Google Colab
Go to:
https://colab.research.google.com/

### Step 2: Upload Files
Upload:
- `student_marks.csv`
- `colab_student_mark_analysis.ipynb`

### Step 3: Run the Notebook
Open the notebook and run all cells one by one.

## Output
The notebook generates:
- Dataset preview

- Subject-wise bar chart visualization
